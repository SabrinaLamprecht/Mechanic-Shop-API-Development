Mechanic Shop API Development

This project is a RESTful API built using Flask, SQLAlchemy, and Marshmallow. It is designed to manage a mechanic shop system with Customers, Mechanics, and Service Tickets.

It features:
--- CRUD operatins
--- Many-to-Many relationship between Service Tickts and Mechanics
--- One-to-Many relationship between Customers and Service Tickets
--- Data serialization & validation using Marshmallow
--- MySQL integration with SQLAlchemy ORM
--- Fully tested endpoints using Postman

Instructions:

1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
4. Configure MySQL Database
5. Initialize the Database
